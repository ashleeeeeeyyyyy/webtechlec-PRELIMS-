HTML (Hypertext Markup Language)

  • Language used to mark up documents(i.e web sites) in the world wide web.

  • Historically used to specify the structure & content of web pages, as well as their presentation

  • Tim Berners-Lee was the primary author of HTML

    • Version History

  • HTML 2.0

  • It is an expansion of HTML 1.0.
  • Has a form-based File upload.
  • It was released in November 25, 1995.

  • HTML 3.2

  • Has HTML 2.0 features such as tables, applets, text flow around images, superscripts and subscripts
  • Released on January 12, 1997

  • XHTML 1.0

  • A new language based on HTML 4.0 that is associated with XML.
  • Was adopted by W3C Recommendation on January 26, 2000

  • XHTML 1.1

  • It is a module-based XHTML.
  • Was adopted by W3C Recommendation on May 2001

  • HTML5

  • Evolved HTML and addresses backward compatibility issues of the HTML.
  • Supplies the needs of modern web applications
  • It was moved to Proposed Recommendation on September 16, 2014

  • HTML5.1

  • Fixed some issues that were left open in HTML5.
  • It was moved to Proposed Recommendation on November 1, 2016

  • HTML5.2

  • It was moved to Proposed Recommendation on December 14, 2017

    • HTML Document Structure

  • HTML VERSION INFORMATION

  • A declaration of what version of HTML is used.

  • HTML ELEMENT

  • The main element that consists the Document head and body.

  • DOCUMENT HEAD

  • Part of the HTML Element that consists of HEAD Element and TITLE Element

  • HEAD Element

  • It contains information about the document like the title of the document

  • TITLE Element

  • Is used by authors to identify the contents of a document

  • DOCUMENT BODY

  • Part of the HTML Element that consists the main content of the document.

  • BODY Element

  • It is the text, image, color found in the Document Body.

  • Element Identifiers

  • These are used to declare elements to be easily distinguished.
  • The two identifiers used in element identifying are "id" and "class"

  • Grouping Elements

  • These are used to put blocks of codes in one container.
  • Example for the grouping elements are the "div" and "span" identifiers

  • Heading Elements

  • It describes the topic of the section it introduces.
  • It is also used to distinguish what is the important and what is less important.
  • The identifiers for the heading elements are h1,h2,h3,h4,h5 and h6 where h1 is the most important and h6 is the less important

  • ADDRESS Elements

  • It is where the contact of the author is found.
  • It is always found in the last part of the bodt element.
